# SkillHub - Landing Page

Landing page publique de la plateforme interne SkillHub (Asteria Group).

## Demo en ligne
https://dravelimbolo.github.io/skillhub/

## Stack
- HTML5 sémantique
- CSS3 (Flexbox, Grid, variables)
- JavaScript Vanilla ES6+ (fetch AJAX, validation formulaire)

## Lancer le projet en local
Servir avec un serveur local (requis pour fetch) :
```
npx serve .
```
